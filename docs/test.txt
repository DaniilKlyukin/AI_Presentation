## Слайд 4. Символьные вычисления с SymPy

**Объявление переменных и функций:**

```python
x = sp.Symbol('x')
f = x**2 * sp.exp(-x)
```

**Упрощение, подстановка, красивая печать:**

```python
sp.simplify((x**2 - 1)/(x - 1))  # -> x + 1
f.subs(x, 2)                     # 4*exp(-2)
sp.init_printing()               # формула в LaTeX-стиле
```

**Пример:** аналитическое преобразование $\frac{x^2-1}{x-1} = x+1$ помогает понять поведение функции до численных расчётов.

## Слайд 5. Пределы функций

**Символьное вычисление пределов:**

```python
x = sp.Symbol('x')
limit1 = sp.limit(sp.sin(x)/x, x, 0)      # -> 1
limit2 = sp.limit((1 + 1/x)**x, x, sp.oo) # -> E
```

Математически:  
$$\lim_{x\to 0} \frac{\sin x}{x} = 1, \quad \lim_{x\to\infty} \left(1+\frac{1}{x}\right)^x = e$$

**Визуализация поведения вблизи точки:**

```python
import numpy as np
import matplotlib.pyplot as plt
x_vals = np.linspace(-1, 1, 500)
y_vals = np.sin(x_vals)/x_vals
plt.plot(x_vals, y_vals)
plt.scatter(0, 1, color='red', zorder=5)
plt.title(r'$\lim_{x\to 0} \frac{\sin x}{x} = 1$')
plt.grid(True)
plt.show()
```

→ График наглядно показывает «устранимую особенность».

## Слайд 6. Производные: дифференцирование

**Символьная производная:**

```python
x = sp.Symbol('x')
f = x**3 - 3*x**2 + 2*x
f_prime = sp.diff(f, x)   # 3*x**2 - 6*x + 2
```

Математически:  
$$f(x)=x^3-3x^2+2x \quad\Rightarrow\quad f'(x)=3x^2-6x+2$$

**Частные производные (для нескольких переменных):**

```python
x, y = sp.symbols('x y')
g = x**2 * y + sp.sin(x*y)
dg_dx = sp.diff(g, x)     # 2*x*y + y*cos(x*y)
dg_dy = sp.diff(g, y)     # x**2 + x*cos(x*y)
```

$$\frac{\partial g}{\partial x}=2xy + y\cos(xy),\quad \frac{\partial g}{\partial y}=x^2 + x\cos(xy)$$

**Градиент** $\nabla g = \left(\frac{\partial g}{\partial x}, \frac{\partial g}{\partial y}\right)$ — Python вычисляет мгновенно.

## Слайд 7. Касательная к графику: от теории к коду

**Уравнение касательной:**  
$$y = f(x_0) + f'(x_0)\,(x - x_0)$$

**Реализация в Python:**

```python
def tangent_line(f, df, x0, x_range):
    x_vals = np.linspace(x_range[0], x_range[1], 200)
    y_vals = f(x_vals)
    y_tan = f(x0) + df(x0)*(x_vals - x0)
    plt.plot(x_vals, y_vals, label='f(x)')
    plt.plot(x_vals, y_tan, '--', label='касательная')
    plt.scatter(x0, f(x0), color='red')
    plt.legend(); plt.grid()
```

**Пример:** $f(x)=x^2$, $x_0=1$ → касательная $y = 2x - 1$.

## Слайд 8. Интегрирование: определённое и неопределённое

**SymPy – аналитические интегралы:**

```python
x = sp.Symbol('x')
indef = sp.integrate(x*sp.exp(x), x)          # (x-1)*exp(x)
definite = sp.integrate(x*sp.exp(x), (x, 0, 1)) # 1
```

$$\int x e^x\,dx = (x-1)e^x + C, \quad \int_0^1 x e^x\,dx = 1$$

**Численное интегрирование (SciPy)** – когда первообразная не берётся:

```python
from scipy import integrate
result, error = integrate.quad(lambda x: np.sin(x)/x, 1e-8, 10)
# интеграл от sin(x)/x от 0 до 10 ≈ 1.658
```

$$\int_0^{10} \frac{\sin x}{x}\,dx \approx 1.658$$

**Площадь под кривой** – мгновенно, без вывода формулы.

## Слайд 9. Визуализация интеграла как площади

```python
x = np.linspace(0, 2, 300)
y = np.exp(-x**2)
plt.plot(x, y, 'b', linewidth=2)
x_fill = np.linspace(0.2, 1.5, 100)
plt.fill_between(x_fill, np.exp(-x_fill**2), alpha=0.3)
plt.title(r'$\int_{0.2}^{1.5} e^{-x^2} dx \approx $' + 
          str(integrate.quad(lambda x: np.exp(-x**2), 0.2, 1.5)[0]))
```

→ Гауссов интеграл $\int e^{-x^2}dx$ (функция ошибок) визуально понятен.

## Слайд 10. Ряды Тейлора: приближение функций

**Разложение в SymPy:**

```python
x = sp.Symbol('x')
f = sp.cos(x)
series = sp.series(f, x, 0, 5)   # 1 - x**2/2 + x**4/24 + O(x**5)
```

$$\cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} + O(x^5)$$

**Сравнение с исходной функцией (численно):**

```python
x_vals = np.linspace(-2, 2, 200)
approx = 1 - x_vals**2/2 + x_vals**4/24
plt.plot(x_vals, np.cos(x_vals), label=r'$\cos x$')
plt.plot(x_vals, approx, '--', label=r'Тейлор (до $x^4$)')
plt.legend()
```

→ Видно, что для малых $x$ приближение отлично, дальше расходится.

## Слайд 11. Решение обыкновенных дифференциальных уравнений

**Задача Коши:**  
$$y' = -2y + t,\quad y(0)=1$$  
Аналитическое решение: $y(t)=\frac{2t-1}{4}+\frac{5}{4}e^{-2t}$, но численно проще.

```python
from scipy.integrate import solve_ivp
def ode(t, y): return -2*y + t
sol = solve_ivp(ode, (0, 5), [1], t_eval=np.linspace(0,5,100))
plt.plot(sol.t, sol.y[0])
plt.title(r"Решение ОДУ $y' = -2y + t$")
```

**Преимущество:** не нужно выводить формулу, достаточно задать правую часть.

## Слайд 12. Поле направлений – геометрия ОДУ

**Построение с помощью `matplotlib.quiver`:**

```python
y, t = np.mgrid[0:5:20j, 0:5:20j]
dy = -2*y + t
dt = np.ones_like(dy)
plt.quiver(t, y, dt, dy, alpha=0.6)
plt.xlabel('t'); plt.ylabel('y')
plt.title(r'Поле направлений для $y'' = -2y + t$')
```

→ Помогает предсказать поведение решений без точной формулы.

## Слайд 13. Многомерный анализ: градиент и экстремумы

**Градиент** $\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}\right)$ – направление наискорейшего роста.  
Поиск минимума $f(x,y) = x^2 + y^2$ с помощью SciPy:

```python
from scipy.optimize import minimize
f = lambda v: v[0]**2 + v[1]**2
res = minimize(f, [10, -5])
print(res.x)  # → [0., 0.]
```

**Визуализация уровней:** контурный график + градиентные стрелки.

```python
X, Y = np.mgrid[-2:2:50j, -2:2:50j]
Z = X**2 + Y**2
plt.contour(X, Y, Z, levels=15)
plt.colorbar()
```
